Which three lines, when inserted independently at line n1, cause the program to print a 0 balance?

---

* A  
> this.amount = 0;  

* B  
> amount = 0;  

* C  
> acct(0);  

* D  
> acct.amount = 0;  

* E  
> acct.getAmount() = 0;  

* F  
> acct.changeAmount(0);  

* G  
> acct.changeAmount(-acct.amount);  

* H  
> acct.changeAmount(-acct.getAmount());
